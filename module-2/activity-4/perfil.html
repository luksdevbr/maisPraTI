BigO Analyzer

The BigO Analyzer is a web-based tool that analyzes the asymptotic complexity (big-O notation) of code snippets. It provides an analysis of time complexity (worst case, best case, and average case) and space complexity. Powered by Google's Gemini AI, this project supports bilingual output (English and Portuguese) and features a responsive design.
